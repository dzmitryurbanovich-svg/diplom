# Thesis Source (LaTeX)

Здесь находятся исходные файлы текста дипломной работы.

## Инструкция по компиляции

Для корректной сборки (включая оглавление и библиографию) необходимо использовать `XeLaTeX` и `BibTeX`.

### Команда для сборки:
```bash
xelatex main.tex
bibtex main
xelatex main.tex
xelatex main.tex
```

### Основные файлы:
- `main.tex`: Главный файл, объединяющий все части.
- `work.tex`: Основная практическая часть (Главы 1-4).
- `refs.bib`: Список литературы.
- `images/`: Папка с графиками и схемами архитектуры.

Результат сборки: `main.pdf`.
