# Demos & Tests

Скрипты для проверки и демонстрации работы системы.

## Описание скриптов

### 1. `demo_engine.py` (Unit Test логики)
Проверяет базовую математику: установку тайла, повороты и правильность работы DSU (завершение города).

### 2. `test_mcp_client.py` (Integration Test)
Проверяет работоспособность MCP-сервера. Имитирует внешние запросы от ИИ по протоколу JSON-RPC.

### 3. `ollama_agent.py` (Full Demo)
Связующее звено между **Ollama** и вашим движком. 
- Позволяет LLM "видеть" поле.
- Позволяет LLM делать ходы, вызывая инструменты сервера.

## Как запускать
Всегда запускайте скрипты из корня проекта, устанавливая `PYTHONPATH`:
```bash
PYTHONPATH=. venv/bin/python demos/ollama_agent.py
```
